# マーケティング戦略

ブログ「AIに詳しい必要はない」（colet-lab.jp / ai-lab）のマーケ全体設計。
最終更新: 2026年4月18日

## ゴール

**法人リード獲得**（研修・伴走・DX支援案件）。
有料記事販売はしない。問い合わせフォーム（toB）／LINE公式（toC）で獲得、法人案件に繋げる。

## ターゲット

- **1次（既存・太い経路）**: 九州博報堂・電通九州・JR九州エージェンシー・自治体 → AI導入研修・DX案件の発注元
- **2次（代理店・自治体経由）**: 地方SMB経営者（顧客のクライアント）
- **3次（広く薄く）**: 全国の「AI導入したいがわからない」SMB

## ファネル（コンテンツフライホイール）

でぐさんの日々のAI実作業1回を、複数チャネルに展開。1原作業 → N成果物。

```
でぐ作業（AI実験）
 ├→ 録画
 ├→ YouTube本編（10〜20分）
 ├→ YouTube Shorts / Instagram Reels（30〜45秒）
 ├→ X / Threads（テキスト切り抜き・画像）
 ├→ ai-lab記事（WordPress）
 └→ ラボの棚の資料（WordPress）
      ↓
  LINE公式 / HPフォーム（リード獲得）
      ↓
   相談・セミナー・法人発注
```

## 主要チャネルと戦略

### 自社HPブログ（colet-lab.jp / ai-lab）
- 週3本ペース
- 記事構造・SEO原則は CLAUDE.md 参照

### YouTube（コレットラボ既存チャンネル）
- 本編：週1本（10〜20分、画面録画+顔出しワイプ）
- Shorts：週3本（30〜45秒）
- 撮影場所：社内、機材：DJI / 一眼 / OBS / Premiere

### X（法人アカウント、フォロワー 950）
- 毎日3〜5本
- Reply誘発で終わる、URL は最初のリプライ、画像推奨、Bookmark誘発

### Threads（法人アカウント、フォロワー 10）
- 毎日1本以上
- 画像付き、問いかけ、URL貼ってOK、投稿＋返信候補セットで生成

### Instagram（法人アカウント、フォロワー 130）
- Reels中心、本編の切り抜き
- 15〜30秒、3秒フック、Share Rate重視

### LINE公式
- 週2本（火12:30 + 金18:30）
- 月初にパスワード配信＋更新情報
- リッチメニュー6枠
